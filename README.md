# eth_service_demo

UDP 车载以太网络报文收发 demo

## 项目简介

本项目为基于 Python 实现的 UDP 车载以太网络报文收发演示（Demo）。可用于学习和测试车载以太网环境下的 UDP 报文发送与接收功能。

## 功能介绍

- 基于 Python 实现 UDP 报文的收发
- 适用于车载以太网消息测试与演示

## 使用方法

1. 克隆本仓库：
   ```bash
   git clone https://github.com/prowerInJava/eth_service_demo.git
   ```
2. 安装依赖（如有 requirements.txt 可直接用 pip 安装）：
   ```bash
   pip install -r requirements.txt
   ```
3. 执行脚本，具体请参考项目内相关 Python 文件。

## 运行环境

- Python 3.x
- 支持 UDP 的网络环境
